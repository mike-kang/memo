## 🔎 BASH_REMATCH란?

Bash에서 `[[ 문자열 =~ 정규식 ]]` 형태로 비교하면,  
**매칭 결과가 자동으로 `BASH_REMATCH` 배열에 저장**됩니다.

형식:

- `BASH_REMATCH[0]` → **정규식에 매칭된 전체 문자열**
    
- `BASH_REMATCH[1]` → 첫 번째 괄호 `()` 그룹
    
- `BASH_REMATCH[2]` → 두 번째 괄호 `()` 그룹
    
- …

```
arg="client-15"

if [[ "$arg" =~ ^client-([0-9]+)$ ]]; then
    echo "전체 매칭: ${BASH_REMATCH[0]}"
    echo "숫자 부분: ${BASH_REMATCH[1]}"
fi

```
